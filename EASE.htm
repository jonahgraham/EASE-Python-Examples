Use the EASE Python Console to run simple Python commands

ease python script shell

Using the console view, run hello world, by pasting the following in. The output will appear in the Console view.

print("Hello World")

The console also allows you to paste into the input box, try to copy and paste this line.

print("Question: Can we extend Eclipse with Python?\nAnswer: Yes, we can")

TIP: You can drag and drop commands to script shell results section

Try some mathematical operations:

2+2

7*8

TIP: Use the up and down arrows in the script shell input box to show history

TIP: Use drop arrow to see history in ...

The script shell also supports multiline input. Try typing the lines below in the script shell. Note you must ensure you add the whitespace at the beginning on the 2 and 3rd lines. Press enter twice to complete the while loop.

for line in range(10):

  print("I love scripting")

Not adding the space will result in an IndentationError being raised such as this:

 

       org.eclipse.ease.ScriptExecutionException:   File "<console>", line 2

             print("poo")

             ^

       IndentationError: expected an indented block

 

Python gives you access to a wealth of libraries. Use import to access a library

import math

math.sqrt(2)

OR

from math import sqrt

sqrt(9)

Check which version of Python you are using with the sys library

import sys

sys.version

sys.platform

sys.path

exercise 2: scripting eclipse: EASE modules

In addition to Python libraries, EASE provides modules. Modules are logical groupings of functionality such as UI, Launch or Debug.  Modules typically package up Eclipse and Java APIs to allow them to be scripted from Python or any other language supported by EASE.

One popular module for controlling the Eclipse user interface is the UI module. Give it a try from the script shell.

loadModule("/System/UI")

showInfoDialog("Hello World")

showInputDialog("What is your name?")

showView("org.eclipse.ease.ui.views.modulesExplorer")

maximizeView("org.eclipse.ease.ui.views.modulesExplorer")

 

#open the console view

showView("org.eclipse.ease.ui.views.modulesExplorer")

Modules Explorer features:

·        Browse modules and functions

·        Links to help

·        Drag to script shell to load them
